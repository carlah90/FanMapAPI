# Colección Postman — FanMapAPI

Esta carpeta contiene la colección utilizada para simular la API del proyecto FanMap mediante Postman.

## Archivos incluidos

- **FanMapAPI.postman_collection.json**  
  Colección con el endpoint GET /kpop/groups y su Example asociado.

- **mockserver.txt**  
  Contiene la URL del Mock Server generado por Postman.

## Mock Server

El Mock Server permite ejecutar solicitudes HTTP sin necesidad de un backend real.
