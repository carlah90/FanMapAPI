# FanMapAPI

Este repositorio contiene el proyecto FanMap, incluyendo:

- Esquema JSON de artistas y grupos K‑pop
- Exportación de datos desde MongoDB Compass
- Colección Postman con endpoint GET /kpop/groups
- Mock Server generado por Postman
- Capturas de evidencia utilizadas en el informe

##  Estructura del repositorio

- /mongo → Datos exportados desde MongoDB
- /postman → Colección Postman + Mock Server
- /docs → Capturas del proyecto
